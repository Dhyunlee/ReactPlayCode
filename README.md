# Getting Started

## before you start

아래 것들이 설치되어있는 지 확인하세요!

- Nodejs 설치

- npm 또는 yarn

## Project installation and start

- Install npm, yarn

### [ npm ]

```bash
   npm install
```

### [ yarn ]

```bash
   yarn install
```

<br/>

## Start project

### [ npm ]

```bash
   npm start
```

### [ yarn ]

```bash
   yarn start
```
